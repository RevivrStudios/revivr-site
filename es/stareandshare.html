Stare&Share

Habla con tus ojos. Convierte una mirada sostenida en conexión instantánea con tu cuidador.

La Experiencia

Comunicación por Permanencia de Mirada

Stare&Share utiliza la mirada sostenida como interacción principal. En lugar de requerir habla, gestos de mano, escritura o toque, el usuario puede concentrarse en un objetivo durante un tiempo establecido para activar un mensaje.

El flujo de permanencia está diseñado para ser tolerante. Un parpadeo corto o un movimiento natural de ojos no debería cancelar el progreso inmediatamente. La aplicación usa un período de gracia para que la interacción se sienta humana en lugar de frágil.

Confirmación Visual

El usuario de Vision Pro ve un anillo de progreso de permanencia claro mientras mira el objetivo. Una vez que se completa la permanencia de mirada, la interfaz cambia de estado para que el usuario sepa que la aplicación está enviando el mensaje.

Mensajería para Cuidadores

Cuando se completa un evento de permanencia, la aplicación envía un mensaje ligero a un dispositivo de cuidador emparejado. El cuidador puede recibir el mensaje en iPhone, iPad o Apple Watch y reconocerlo con una acción simple de "Visto".

El usuario de Vision Pro recibe entonces una confirmación, cerrando el bucle de comunicación.

Conciencia de Objetos Físicos y Reconocimiento en el Dispositivo

La aplicación está destinada a comunicar atención intencional, no miradas accidentales. Valida que el usuario esté mirando un objeto físico real o una superficie en lugar de espacio vacío utilizando detección del mundo y comprensión de la escena.

Stare&Share está diseñado para identificar el objeto utilizando inteligencia en el dispositivo donde sea posible. El objetivo es enviar contexto útil al cuidador, como "vaso de agua", "teléfono" o "frasco de medicamento", sin comprometer la privacidad.

Diseño con Privacidad Primero

  • Sin datos de mirada sin procesar almacenados o transmitidos.
  • Clasificación de objetos en el dispositivo siempre que sea posible.
  • Fotos o fotogramas de cámara solo se utilizan cuando los permisos de plataforma aprobados lo permiten explícitamente.
  • Emparejamiento estrictamente limitado a dispositivos de cuidadores de confianza.
