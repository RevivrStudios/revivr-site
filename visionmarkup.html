VisionMarkUp

Vision Markup allows for fast, fun, and easy markups using your hands or the Logitech Muse pen directly inside the Vision Pro headset, all seamlessly shared with your entire team.

Drawing Tools

  • Full PencilKit integration with pen, marker, and highlighter
  • Precision input via Logitech Muse stylus or Apple Pencil
  • Left-hand touch support for direct annotation
  • Trackpad compatibility for detailed work
  • Real-time color and tool selection
  • Image Crop
  • Grid Mode

Image Workflows

  • Drag and drop images from Photos or Files
  • PhotosPicker integration for seamless import
  • Rotate images in 90° increments with dedicated editor
  • High-quality JPEG export (optimized compression)
  • System share sheet for universal distribution
  • Layer Separation
  • Common Premade Shapes

Professional Features

  • Spatial canvas for immersive annotation
  • Clean, distraction-free interface
  • Instant tool switching with no lag
  • Privacy-first: all processing on-device
  • No cloud sync, no data collection

Ideal For

  • Design review and feedback
  • Screenshot annotation
  • Photo markup for documentation
  • Spatial computing workflows
  • Creative professionals on Vision Pro

Built for visionOS

Vision Markup is built specifically for visionOS, leveraging Apple's Human Interface Guidelines to deliver a native, intuitive experience. Whether you're marking up designs, annotating screenshots, or adding notes to images, Vision Markup provides the tools you need in a spatial environment.

Import your image, choose your tool, and start creating.
