Stare&Share

Speak with your eyes. Turn a steady gaze into instant caregiver connection.

The Experience

Gaze Dwell Communication

Stare&Share uses sustained gaze as the primary interaction. Instead of requiring speech, hand gestures, typing, or touch, the user can focus on a target for a set duration to trigger a message.

The dwell flow is designed to be forgiving. A short blink or natural eye movement should not immediately cancel progress. The app uses a grace period so the interaction feels humane instead of brittle.

Visual Confirmation

The Vision Pro user sees a clear dwell progress ring while they are looking at the target. Once the gaze dwell completes, the interface changes state so the user knows the app is sending the message.

Caregiver Messaging

When a dwell event completes, the app sends a lightweight message to a paired caregiver device. The caregiver can receive the message on iPhone, iPad, or Apple Watch and acknowledge it with a simple "Seen" action.

The Vision Pro user then receives confirmation, closing the communication loop.

Physical Object Awareness & On-Device Recognition

The app is meant to communicate intentional attention, not accidental glances. It validates that the user is looking at a real physical object or surface rather than empty space using world sensing and scene understanding.

Stare&Share is designed to identify the object using on-device intelligence where possible. The goal is to send useful context to the caregiver, such as "water glass," "phone," or "medication bottle," without compromising privacy.

Privacy-First Design

  • No raw gaze data stored or transmitted.
  • On-device object classification wherever possible.
  • Photos or camera frames only used when explicitly supported by approved platform permissions.
  • Pairing strictly limited to trusted caregiver devices.
