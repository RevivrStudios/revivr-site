How to Use Track Stash

  1. Start with Quick Stash when you need to capture an idea fast. Record the sound, take gear photos, and add a few notes.
  2. When the idea is worth keeping, create a full session from the Quick Stash or start a new session from the Sessions tab.
  3. Add the gear used in the session from your Gear Library. Save the patch name, role, template settings, and photos for each piece of gear.
  4. Build the signal chain so you know how everything was connected.
  5. Add audio, video, photos, mixes, and notes as the track develops.
  6. Use Recall when you want to rebuild the setup later. Review the signal flow, check off connections, and open gear patch details to restore the sound.
