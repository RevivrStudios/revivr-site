# Revivr Studios Website

A premium, dark-themed portfolio site for Revivr Studios, built with Vite and Vanilla JavaScript.

## Setup

1. **Install Node.js**: Ensure Node.js is installed on your system.
2. **Install Dependencies**:
    ```bash
    npm install
    ```
3. **Run Development Server**:
    ```bash
    npm run dev
    ```
    Open the link provided (usually `http://localhost:5173`) to view the site.

## Build for Production

1. Run the build command:
    ```bash
    npm run build
    ```
2. The output will be in the `dist` folder, ready for deployment.

## Project Structure

- `index.html`: Main landing page with App Grid.
- `mission.html`: Company mission statement.
- `founder.html`: Founder biography.
- `support.html`: Support contact info.
- `webxr.html`: A-Frame WebXR playground.
- `src/css`: Global and layout styles.
- `src/js`: Main JavaScript logic.
